Pengertian WEB

Web atau World Wide Web (WWW) adalah sistem informasi berbasis internet yang memungkinkan pengguna mengakses berbagai konten seperti teks, gambar, video, dan dokumen melalui browser. Web menggunakan protokol HTTP/HTTPS dan terdiri dari halaman yang saling terhubung melalui hyperlink.

Cara Kerja WEB

* Pengguna memasukkan URL di browser.
* Permintaan dikirim ke server web.
* Server mengirimkan halaman dalam bentuk HTML, CSS, dan JavaScript.
* Browser menerjemahkan dan menampilkan halaman web.

Jenis-Jenis WEB

* Web Statis → Kontennya tetap dan hanya bisa diubah oleh pemiliknya.
* Web Dinamis → Kontennya dapat berubah sesuai interaksi pengguna (misalnya media sosial, e-commerce).
* Web 1.0 → Versi awal, hanya bisa membaca informasi.
* Web 2.0 → Lebih interaktif, pengguna bisa berkontribusi (misalnya blog, forum).
* Web 3.0 → Berbasis AI, blockchain, dan lebih personal.

Komponen Utama WEB

* Domain → Alamat unik web (contoh: www.google.com).
* Hosting → Server tempat menyimpan file web.
* HTML → Struktur dasar halaman web.
* JavaScript → Untuk interaksi dan fungsionalitas.
* CSS → Untuk desain dan tampilan web.
